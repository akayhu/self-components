### 安裝

```bash
npm install
```

## Package Script 說明

運行 react app
> 本專案框架為create-react-app (CSR)

```bash
npm start
```

運行 storybook

```bash
npm run storybook
```

運行 next 
> next 只是嘗試安裝框架，不確定是否會與 create-react-app(CSR) 框架有所衝突與重複性框架架構
> 待測試或確認

```bash
npm run start-next
```

src 底下所有程式碼自動排版

```bash
npm run prettier
```
